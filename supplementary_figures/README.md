This folder contains supplementary figures in the manuscript "GUT MICROBIOME METABOLITES EXHIBIT REGORAFENIB-LIKE TRANSCRIPTIONAL SIGNATURES IN COLORECTAL CANCER: A COMPUTATIONAL MICROBIOME PHARMACOLOGY APPROACH"

# Supplementary Figures

## Supplementary Figure S1a
File: Fig_S1a_score_scatter.pdf  
Title: Baseline versus longitudinal consensus regorafenib mimic scores.  
Caption: Each point represents one metabolite feature. Red points indicate features corresponding to metabolites selected for experimental testing. The dashed line indicates perfect concordance between baseline and consensus scores; Spearman rank correlation was ρ = 0.235.

## Supplementary Figure S1b
File: Fig_S1b_purdue_heatmap.pdf  
Title: Regorafenib mimic scores for experimentally tested metabolites across baseline and longitudinal consensus signatures.  
Caption: Heatmap shows Spearman correlation-based regorafenib mimic scores for seven IBDMDB metabolite features corresponding to six experimentally tested metabolites; inosine was detected as two separate metabolite features. Positive values indicate regorafenib-like directionality, whereas negative values indicate regorafenib-opposing directionality. Asterisks indicate nominal significance at p < 0.05.
