This folder contains Supplemental Tables in the manuscript "GUT MICROBIOME METABOLITES EXHIBIT REGORAFENIB-LIKE TRANSCRIPTIONAL SIGNATURES IN COLORECTAL CANCER: A COMPUTATIONAL MICROBIOME PHARMACOLOGY APPROACH"

# Supplementary Tables

## Supplementary Table S1a
File: Table S1a - Top regorafenib-like metabolites.csv  
Title: Top regorafenib-like metabolites.  
Description: Metabolites consistently positively correlated with the regorafenib transcriptional signature across unadjusted, sex-adjusted, and disease-adjusted analyses.

## Supplementary Table S1b
File: Table S1b Top regorafenib-opposing metabolites.csv  
Title: Top regorafenib-opposing metabolites.  
Description: Metabolites consistently negatively correlated with the regorafenib transcriptional signature across unadjusted, sex-adjusted, and disease-adjusted analyses.

## Supplementary Table S2a
File: Table S2a - KEGG pathway enrichment results for regorafenib-like metabolites.csv  
Title: KEGG pathway enrichment results for regorafenib-like metabolites.  
Description: FGSEA results for pathways enriched among genes associated with regorafenib-like metabolites, including normalized enrichment score and adjusted p-value.

## Supplementary Table S2b
File: Table S2b - KEGG pathway enrichment results for regorafenib-opposing metabolites.csv  
Title: KEGG pathway enrichment results for regorafenib-opposing metabolites.  
Description: FGSEA results for pathways enriched among genes associated with regorafenib-opposing metabolites, including normalized enrichment score and adjusted p-value.

## Supplementary Table S3
File: Table S3 - Consensus sensitivity results for experimentally tested metabolites.csv  
Title: Consensus sensitivity results for experimentally tested metabolites.  
Description: Baseline and longitudinal consensus mimic scores for seven IBDMDB metabolite features corresponding to six metabolites selected for experimental testing.

## Supplementary Table S4
File: Table S4 - Beta-glucuronidase annotation of regorafenib-associated taxa.csv  
Title: β-glucuronidase annotation of regorafenib-associated taxa.  
Description: Regorafenib-associated taxa annotated for β-glucuronidase potential using UniProt evidence, including EC 3.2.1.31.

## Supplementary Table S5a
File: Table S5a - Kaplan-Meier/Cox results for mimic-associated genes.csv  
Title: Survival analysis results for genes associated with regorafenib-like metabolites.  
Description: Kaplan-Meier and Cox proportional hazards results for genes associated with top-ranked metabolites.

## Supplementary Table 54b
File: Table S5b - Kaplan-Meier/Cox results for opposite-associated genes.csv  
Title: Survival analysis results for genes associated with regorafenib-opposing metabolites.  
Description: Kaplan-Meier and Cox proportional hazards results for genes associated with top-ranked metabolites.
